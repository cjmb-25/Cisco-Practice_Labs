Router-3-configuration:
-----
enable
configure terminal
no ip domain-lookup
hostname Router-3
banner motd $
Unauthorized access prohibited!

Lab Credentials
enable: Epass
console: Cpass
vty: Vpass $

enable secret Epass

line console 0
password Cpass
login
exit

line vty 0 4

exit

no shutdown
exit

no shutdown
exit
